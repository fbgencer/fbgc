957 /*
958  [FBGC Malloc algorithm]
959 
960  In this section, there are different situations to allocate memory efficiently.
961 
962  Requested size : size, available size will be asize
963 
964  Memory structure contains object memory pools which are allocated spaces in the heap
965  fbgc_memory_pool contains *data,*tptr, size and next pointer.
966  Different pools are connected each other like a single linked list with the next pointer.
967  Size holds the size of the memory pool. Each pool creates a single pool which is called as object pool in fbgc.
968 
969  This algorithm allocates new spaces, runs gc and so on.
970 
971  1.Start iterating inside memory pools. Check their size and compare it with the requested size.
972  While iterating in each list we will check the following condition.
973  o) Requested size is smaller than the object pool,
974  Calculate the empty space that we have by using tptr(traverser)
975  if we have enough space return the pointer of this space
976  [---------------------------------| ]
977  ^ ^ ^
978  memory_pool_data memory_pool_tptr memory_pool_data + memory_pool_size
979 
980  o) We iterated in each memory pools and requested size is bigger than any of them.
981  Here new allocation takes place and due to the state we directly use goto for allocation.
982 
983  o) Requested size is not bigger than the memory pools. However, there is no enough space to give the requsted.
984  We will check our empty lists, which is the list from previous garbage collection running. If there is enough space in there
985  no need to run gc in this moment.
986  If this also fails, the worst case scenario happens for this function: running garbage collector!
987  Garbage collector will run, basically it will tell all the variables to mark their objects
988  We will sweep in each part of the memory pools one by one to create empty memory lists.
989  Again empty memory list will be searched and if there is NO space we will allocate new memory.
990  New memory is basically another memory pool for the requested size.
991
